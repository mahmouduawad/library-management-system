# Library Management System 📚

A simple console-based Library Management System developed using Java and Object-Oriented Programming (OOP). The system allows users to manage books and borrowers efficiently through a menu-driven interface.

## 🚀 Features

### 📖 Book Management
- Add new books
- Update existing book details
- Delete books
- Search for a book by ID
- Display all available books

### 👤 Borrower Management
- Add borrower records with book borrowing details
- View all borrowers and their borrowed books

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList for data storage
- Scanner for user input

## 💡 Concepts Applied
- Classes and Objects
- Encapsulation
- Static Methods
- Lists (ArrayList)
- Menu-driven program structure
